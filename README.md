# Weekly Topics for AZ-400 Certification Study Guide

This document outlines a weekly study plan for the AZ-400 certification, allocating 3 hours on Saturdays and 3 hours on Sundays. Each week includes theoretical and practical components. Theoretical topics are developed in separate markdown files for easy navigation.

## Week 1: Introduction and DevOps Fundamentals
- [Theory: Introduction and DevOps Fundamentals](week-1-theory.md)
- Practice:
  - Create a free Azure DevOps account
  - Explore the interface and create a sample project
  - Navigate Boards, Repos, Pipelines, Test Plans, and Artifacts

## Week 2: Version Control with Git in Azure DevOps
- [Theory: Version Control with Git](week-2-theory.md)
- Practice:
  - Create and clone repositories in Azure DevOps
  - Perform commits, push, pull, and merge
  - Configure branch policies and pull requests

## Week 3: Continuous Integration (CI)
- [Theory: Continuous Integration](week-3-theory.md)
- Practice:
  - Create a basic CI pipeline
  - Configure automatic triggers
  - Run automated tests in the pipeline

## Week 4: Continuous Delivery (CD) and Deployment
- [Theory: Continuous Delivery and Deployment](week-4-theory.md)
- Practice:
  - Configure a CD pipeline
  - Deploy an application to Azure App Service
  - Implement manual approvals

## Week 5: Security and Compliance in DevOps
- [Theory: Security and Compliance](week-5-theory.md)
- Practice:
  - Use Azure Key Vault in pipelines
  - Configure access policies and auditing

## Week 6: Monitoring, Feedback, and Optimization
- [Theory: Monitoring and Feedback](week-6-theory.md)
- Practice:
  - Set up alerts and dashboards
  - Analyze deployment metrics and logs

## Week 7: Comprehensive Practice and Exam Simulation
- Practice:
  - Review official Microsoft Learn labs
  - Build a complete pipeline: CI + CD + monitoring
  - Take a practice exam and review weak topics
